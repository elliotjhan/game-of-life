# Conway's Game of Life

A modern implementation of **Conway's Game of Life** built with **Next.js** and **TypeScript**.

## Features

- Built with Next.js
- Written in TypeScript
- Interactive Game of Life simulation
- Fast development with hot reload

## Getting Started

### Prerequisites

- Node.js
- npm or Yarn

### Installation

Clone the repository:

```bash
git clone <your-repository-url>
cd <your-project-folder>
```

Install dependencies:

```bash
npm install
```

or

```bash
yarn
```

## Running the Application

Start the development server:

```bash
npm run dev
```

or

```bash
yarn dev
```

Open your browser and navigate to:

```
http://localhost:3000
```

## About Conway's Game of Life

Conway's Game of Life is a cellular automaton devised by mathematician **John Conway**. The simulation evolves over discrete generations according to four simple rules based on the state of neighboring cells:

- Any live cell with fewer than two live neighbors dies.
- Any live cell with two or three live neighbors survives.
- Any live cell with exactly three live neighbors becomes a live cell.
- Any live cell with more than three live neighbors dies.

Despite these simple rules, the simulation produces surprisingly complex and fascinating patterns.
